Kuali Financial System 1.0

This initial release of the Kuali Financial System (KFS) is the first in a series of planned releases in an effort to provide a richly featured financial system for use by higher education institutions.  The KFS will meet GASB and FASB standards, enable a strong control environment, and provide a complete financial system. 

KFS Release 1.0 contains:

  • Chart of Accounts
  • Financial Transactions
  • General Ledger
  • Kuali Nervous System (KNS) - a core technical module composed of reusable code components that provide common pieces of functionality. 
  • Kuali Enterprise Workflow (KEW) - a general-purpose electronic routing infrastructure or "workflow engine".

Implementation Requirements

Central Authentication Service - The version of the Central Authentication Service (CAS) shipped with KFS 1.0 currently does not require passwords.  Implementing institutions will need to configure CAS to require passwords in order to activate secure authentication to the application

Encryption Service - KFS 1.0 uses the Demonstration Encryption Framework to secure and encrypt sensitive data.  Implementing institutions will need to decide whether to use the delivered encryption framework or another encryption framework of their choosing.  A unique key will need to be established as well
