The Kuali Financial System (KFS) is the result of a community source project to develop an integrated set of financial services for higher education. Starting with the Indiana University Financial Information System (IU FIS), the Kuali Project is developing financial modules in distributed Web applications, with open source tools, for access in everyday browsers. Institutions may implement only the functionality they choose, and add or remove modules as necessary.

Kuali is a community source project that orchestrates the functional and technical expertise of its partner institutions.

Working together in collaborative teams, Kuali members have analyzed, designed, created, and provided all of the services presented to you in the Kuali Test Drive. If you are interested in how these services can address the business operations of your organization, please Contact Us.

And no, we did not invent the word Kuali: "kitchen wok - humble utensil which plays an important role in a successful kitchen."
